# 多线程-蒙特卡洛方法计算π

## 算法思想
蒙特卡洛方法计算π是基于几何概率和大数定律的统计模拟方法。核心思想是：通过随机采样来估计难以直接计算的数学常数。

## 算法原理

### 几何基础
考虑一个边长为2的正方形，其面积为：$4$，在这个正方形内画一个半径为1的内切圆，其面积为：$\pi$

圆的面积与正方形面积之比为：$\frac{\pi}{4}$

### 概率解释：
如果我们在正方形内随机投点，那么点落在圆内的概率为：
$$P(\text{点在圆内}) = \frac{\text{圆的面积}}{\text{正方形的面积}} = \frac{\pi}{4}$$

因此：
$$\pi = 4 \times P(\text{点在圆内}) = 4 \times \frac{\text{落在圆内的点数}}{\text{总点数}}$$
